# Robin 🌿

**Sustainability Hackathon 2025 — Exec Choice Award Winner**

Robin is an AI-powered smart waste assistant that helps users dispose of waste correctly in real time. Present an item in front of the camera, and Robin tells you exactly which bin it belongs in, reducing contamination and keeping waste out of landfill.

Built in under 24 hours by a team of 5 as part of a sustainability-themed hackathon, Robin placed and won the Exec Choice Award.

---

## The Problem

Waste contamination is a major and underappreciated environmental issue. When recyclables are thrown in the wrong bin, entire loads get sent to landfill. Robin makes it effortless to do the right thing.

## Features

- AI-powered waste classification via API integration
- Simple, accessible interface for quick lookups
- FAQ page with waste disposal guidance
- Designed with scalability in mind — future iterations could integrate with solar-powered smart bin hardware in public spaces

## Tech Stack

- HTML, CSS, JavaScript
- AI classification API
- Vanilla frontend, no frameworks required
